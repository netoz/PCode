Beneficios

Criptografía y firmas digitales

Garantizan la confidencialidad, integridad y autenticidad de los datos y por consecuencia el NO REPUDIO de los mismos

Soberanía de datos

El usuario es el custodio de sus datos personales (en apego al reglamento GDRP)

Protección contra el robo de identidad

Protección contra falsificación y alteración de documentos

Disponibilidad

Disponibilidad inmediata para el reemplazo y reexpedición, validación a través de dispositivos móviles (iOS y Android)

Durabilidad

Larga durabilidad del documento por una prolongada vida útil del mismo

Solución offline

No requiere conectividad a internet o datos móviles para la validación del documento

Procesamiento distribuido

La verificación es 1:1 al interior del mismo dispositivo, la concurrencia y el uso de servidores centralizados se evitan, logrando una mayor eficiencia

Solución multiplataforma

La validación de PERSONAL CODE está disponible en varias plataformas.
